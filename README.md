# TP Android – Fragments

Ce projet présente un exemple simple d'utilisation des **Fragments** dans une application Android.  
Il permet d'afficher alternativement deux fragments différents à l’aide de deux boutons dans la même activité.

---

## Objectifs

- Comprendre le rôle des **fragments**.
- Manipuler les transactions via `FragmentManager` et `FragmentTransaction`.
- Charger des fragments avec `add()` et `replace()`.
- Utiliser un conteneur dédié (`FragmentContainerView` ou `FrameLayout`).

## Fonctionnement

1. L'activité principale contient deux boutons :  
   - **Fragment 1**  
   - **Fragment 2**

2. Lorsque l'utilisateur clique sur un bouton, l'application remplace le fragment affiché dans le conteneur par :
   - `FragmentOne`  
   - `FragmentTwo`

## Technologies utilisées

- Android Studio
- Java
- AndroidX Fragments
- FragmentContainerView ou FrameLayout
